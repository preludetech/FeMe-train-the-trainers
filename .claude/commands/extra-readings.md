Read `6-activities.md` in the current directory and add a "Further Reading" section at the end (but before the Glossary if one exists). Write the result to `7-readings.md`.

## Format

```markdown
## Further Reading

- Author(s) (Year). *Title*. Publisher/Source. Brief one-sentence note on why this resource is useful.
```

## Guidelines

- Include 2–5 recommended resources for readers who want to go deeper
- Follow the **references** skill for formatting (TASL format where applicable)
- Prioritise:
  - Freely accessible resources (open access, Creative Commons) where possible
  - Foundational and widely respected sources over obscure ones
  - Resources that are practical and readable, not dense academic papers (unless genuinely accessible)
- If you are not confident a resource exists or is freely available, leave a note saying `TODO: CHECK RESOURCE`
- Do not invent references — only include resources you are confident exist
- Use web research to verify URLs and availability where possible
- Do not modify the rest of the content — only add the Further Reading section

## Report

At the end, report any problems and what you have done. If any resources need to be checked then state that clearly.
