# Train the trainers

This repo contains content that is to be included in a course that is currently under development. 

This repo is temporary, the final course will be under a [CC BY 4.0 license](https://creativecommons.org/licenses/by/4.0/?ref=chooser-v1) when released but is still very much a work in progress.

If you find this repo and want to make use of the content then either:

- wait until it's finished
- get in touch: hello@prelude.tech 


# How the presentations work 

If you want to run the presentations then:

```
cd presentation

# install the dependencies. You only need to do this once
npm i

# Now you can launch the presentations individually

npm run demo      # launch a feature demo 
npm run session1  # launch the slides for session 1 
npm run session2  # launch the slides for session 2
```
